-
- $\forall$ every, any, all, forall, arbitraty
	- needs $\rightarrow$
- $\exists$ there, some
	- need $\wedge$
-
- if p, q
	- $p \rightarrow q$
- only if p, q
	- $q \rightarrow p$
- iff p, q
	- $p \leftrightarrow q$
- unless p, q
	- $\neg p \rightarrow q$
- because p, q
	- $p \rightarrow q$
- to do p, q is necessary (or need to q)
	- $p \rightarrow q$
- p is sufficient for q
	- $p \rightarrow q$
- q is necessary for p
	- $p \rightarrow q$
